## 🚀 部署方式

### 自动部署（推荐）
已配置 GitHub Actions 工作流：
```bash
# 推送 wiki 目录自动部署
git add wiki/
git commit -m "docs: 更新 Wiki"
git push
```

### 手动部署
```bash
# 克隆 Wiki 仓库
git clone https://github.com/andjie98/Slimefun4.wiki.git

# 复制文档
cp wiki/*.md Slimefun4.wiki/

# 提交推送
cd Slimefun4.wiki
git add .
git commit -m "Initial Wiki"
git push
```
