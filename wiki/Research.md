## 🎯 研究系统简介

### 什么是研究系统？

研究系统是 Slimefun4 的核心进度机制，玩家需要通过消耗经验值来解锁各种物品和配方的制作权限。

### 研究系统的作用

1. **进度控制** - 防止玩家过早获得强力物品
2. **游戏平衡** - 确保合理的科技发展顺序
3. **成就感** - 提供清晰的进度反馈
4. **探索动力** - 鼓励玩家收集经验和资源

### 研究基础机制

```
消耗经验值 → 解锁研究 → 获得制作权限 → 制作物品 → 继续研究
```

### 研究界面

#### 打开研究指南
1. 手持研究指南
2. 右键点击
3. 查看研究树

#### 界面元素
- **研究图标** - 代表要研究的物品
- **研究等级** - 显示所需等级
- **经验消耗** - 显示需要的经验值
- **解锁状态** - 显示是否已解锁

## 🔓 研究解锁机制

### 研究解锁流程

#### 步骤 1: 准备经验值
- 确保有足够的经验等级
- 建议至少 3-5 级用于研究

#### 步骤 2: 打开研究指南
- 手持研究指南
- 右键点击打开界面

#### 步骤 3: 选择研究项目
- 浏览可研究的物品
- 查看所需经验和等级

#### 步骤 4: 开始研究
- 点击研究图标
- 确认消耗经验值
- 等待研究完成

#### 步骤 5: 研究成功
- 获得制作权限
- 解锁相关配方
- 可制作该物品

### 研究消耗

#### 经验值消耗
- **基础物品**: 1-5 级经验
- **中级物品**: 5-15 级经验
- **高级物品**: 15-30 级经验
- **顶级物品**: 30-50 级经验

#### 等级要求
- 某些物品有最低等级要求
- 必须先研究前置物品
- 遵循科技树顺序

### 研究限制

#### 硬性限制
1. **等级不足** - 无法研究高等级物品
2. **经验不足** - 无法支付研究费用
3. **前置未解锁** - 需要先研究相关物品

#### 软性限制
1. **研究顺序** - 建议按顺序研究
2. **资源限制** - 制作材料不足
3. **科技水平** - 机器未解锁

### 特殊研究机制

#### 自动解锁
某些物品在特定条件下自动解锁：
- 完成特定成就
- 达到特定等级
- 首次登录奖励

#### 隐藏研究
部分隐藏研究项目：
- 不显示在研究树中
- 需要特殊条件触发
- 通常是彩蛋物品

#### 研究重置
```bash
# 管理员命令
/sf research reset <player>
```
- 清空玩家所有研究
- 返还部分经验值
- 谨慎使用

## 💡 研究技巧与策略

### 高效研究策略

#### 1. 优先级规划

**第一阶段：生存基础 (等级 1-10)**
- ✅ 基础工具
- ✅ 基础护甲
- ✅ 小型背包
- ✅ 电动熔炉

**第二阶段：自动化 (等级 10-20)**
- ✅ 粉碎机
- ✅ 货物节点
- ✅ 自动化合成室
- ✅ 中型背包

**第三阶段：高级科技 (等级 20-30)**
- ✅ 电动工具
- ✅ 喷气背包
- ✅ 护身符
- ✅ 高级机器

**第四阶段：完美毕业 (等级 30-38)**
- ✅ 所有剩余物品
- ✅ 收藏物品
- ✅ 特殊物品

#### 2. 经验值管理

**快速获得经验**:
- 建造刷怪塔（推荐蜘蛛或史莱姆）
- 大量烧炼矿物
- 钓鱼获取经验瓶
- 村民交易

**经验值保存**:
- 不要随意死亡
- 使用经验瓶存储
- 避免无意义的研究

#### 3. 资源优化

**研究前准备**:
```
1. 收集足够材料
2. 确保经验充足
3. 解锁相关机器
4. 准备存储空间
```

**避免浪费**:
- 不要研究不需要的物品
- 优先研究实用物品
- 考虑后续升级路线

### 研究技巧

#### 1. 批量研究技巧

**准备工作**:
- 收集大量经验瓶
- 打开研究指南
- 规划研究顺序

**执行步骤**:
```
1. 连续点击研究项目
2. 快速确认
3. 不要中断
4. 记录已研究物品
```

#### 2. 经验值效率

**最佳实践**:
- 使用经验护身符（+30% 经验）
- 在幸运护身符作用下刷怪
- 批量烧炼矿物
- 利用刷怪塔高效升级

**避免低效**:
- ❌ 单独研究每个物品
- ❌ 没有规划就研究
- ❌ 研究后立即使用
- ❌ 频繁死亡丢失经验

#### 3. 研究顺序优化

**推荐顺序原则**:
1. 生产工具优先
2. 存储设备次之
3. 机器设备随后
4. 装饰物品最后

**具体顺序示例**:
```
工具 → 背包 → 基础机器 → 运输系统 → 
护甲 → 高级机器 → 护身符 → 飞行装备 → 
收藏物品
```

### 常见问题与解决

#### Q1: 经验值不够怎么办？

**解决方案**:
1. 建造刷怪塔
2. 大量采矿烧炼
3. 钓鱼获取经验瓶
4. 完成成就获得经验
5. 村民交易

#### Q2: 研究顺序混乱怎么办？

**解决方案**:
1. 查看本指南的推荐顺序
2. 重置研究（管理员）
3. 重新规划路线
4. 参考其他玩家经验

#### Q3: 某些物品无法研究？

**可能原因**:
- 等级不足
- 前置物品未解锁
- 经验值不够
- 物品被禁用

**解决方案**:
1. 提升等级
2. 研究前置物品
3. 收集更多经验
4. 检查配置文件

#### Q4: 研究后无法制作物品？

**可能原因**:
- 缺少材料
- 机器未解锁
- 配方错误
- 世界限制

**解决方案**:
1. 收集所需材料
2. 解锁相关机器
3. 查看正确配方
4. 检查世界 whitelist

## 🎮 研究系统配置

### config.yml 配置

```yaml
# 研究相关配置
research-ranks:
  - Chicken          # 等级 1
  - Cow              # 等级 2
  - Spider           # 等级 3
  # ... 完整等级列表

# 研究解锁设置
allow-free-creative-research: true  # 创造模式免费研究
research-unlock-fireworks: true     # 解锁时播放烟花
research-give-fireworks: true       # 给予烟花奖励
```

### 自定义研究等级

管理员可以在配置文件中自定义研究等级：

```yaml
research-ranks:
  - 新手
  - 学徒
  - 专家
  - 大师
  - 传奇
```

### 权限设置

```yaml
# 研究相关权限
slimefun.cheat.researches:
  description: 允许作弊研究
  default: op
```
