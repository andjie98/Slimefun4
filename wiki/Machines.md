## ⚡ 能源生成机器

### 核反应堆 (Reactor)

最强大的能源生成设备，使用铀燃料棒产生大量能源。

**特点**:
- 极高能源产出
- 需要铀燃料棒
- 产生放射性废料
- 需要反应堆接口控制

**使用方法**:
1. 建造反应堆结构（3x3x3）
2. 在底部放置反应堆接口
3. 通过接口添加燃料棒
4. 使用红石信号控制运行状态

**注意事项**:
- ⚠️ 反应堆过热会爆炸！
- 需要足够的冷却系统
- 定期清理放射性废料

### 太阳能发电机 (Solar Generator)

环保的能源设备，白天自动发电。

**特点**:
- 无需燃料
- 只在白天工作
- 低维护成本
- 适合初期使用

**能源产出**: ~5-10 J/t（取决于天气和时间）

### 生物燃料发电机 (Biofuel Generator)

使用生物燃料产生能源。

**燃料类型**:
- 生物燃油
- 植物油脂
- 其他可燃液体

**能源产出**: 中等，持续稳定

## 🔨 基础加工机器

### 电动熔炉 (Electric Furnace)

高效的矿物冶炼设备。

**特点**:
- 比传统熔炉快 2 倍
- 可自动输入输出
- 支持多个物品同时冶炼

**配方示例**:
```
铁矿石 → 铁锭
金矿石 → 金锭
沙子 → 玻璃
```

**能源消耗**: 中等
**处理速度**: 快速

### 电动铸锭厂 (Electric Ingot Factory)

将金属加工成锭状。

**特点**:
- 处理各种金属粉末
- 生产标准金属锭
- 支持合金制作

**可处理材料**:
- 铁粉 → 铁锭
- 金粉 → 金锭
- 铜粉 → 铜锭
- 锡粉 → 锡锭

### 电动粉末清洗机 (Electric Dust Washer)

提纯矿物粉末的高级设备。

**特点**:
- 提高矿物产出率
- 去除杂质
- 需要水源

**处理流程**:
```
脏粉末 → 清洗 → 纯净粉末 → 冶炼 → 金属锭
```

**产出提升**: 1 个矿石 = 2-3 个金属锭

### 粉碎机 (Ore Crusher)

将矿石粉碎成粉末。

**特点**:
- 1 矿石 = 2 粉末
- 配合粉末清洗机使用
- 大幅提高矿物产出

**可处理矿石**:
- 所有金属矿石
- 石英
- 萤石

**能源消耗**: 中等
**处理速度**: 中等

### 压缩机 (Compressor)

高压压缩材料的设备。

**用途**:
- 煤炭 → 钻石
- 木炭 → 煤炭块
- 其他压缩配方

**特殊配方**:
```
8 煤炭 + 高压 → 钻石
```

**能源消耗**: 高
**处理速度**: 慢

## 🏭 高级制造设备

### 自动化合成室 (Automated Crafting Chamber)

自动执行工作台合成配方。

**特点**:
- 支持所有工作台配方
- 可编程配方
- 自动输入原料
- 自动输出成品

**使用方法**:
1. 放置合成室
2. 使用配置界面设置配方
3. 连接货物输入节点
4. 连接货物输出节点

**能源消耗**: 中等
**处理速度**: 取决于配方复杂度

### 压力舱 (Heated Pressure Chamber)

高温高压环境下处理材料。

**用途**:
- 制作高级合金
- 处理特殊材料
- 合成高科技部件

**特殊配方**:
- 强化合金
- 碳纤维
- 高级电路板

**能源消耗**: 非常高
**处理速度**: 慢

### 炼油厂 (Refinery)

将原油精炼成各种产品。

**产品列表**:
- 汽油（燃料）
- 柴油（燃料）
- 润滑油（工业用）
- 塑料（材料）

**处理流程**:
```
原油 → 精炼 → 各种石油产品
```

**能源消耗**: 高
**处理速度**: 中等

### 采油机 (Oil Pump)

从地下抽取石油。

**特点**:
- 需要找到石油矿点
- 持续抽取
- 需要电力驱动
- 存储容量有限

**使用方法**:
1. 使用地质探测器找到石油
2. 在石油点上放置采油机
3. 连接电源和储罐
4. 启动机器

**抽取速度**: 中等
**存储容量**: 1000mB

### 冷冻机 (Freezer)

将液体固化的设备。

**用途**:
- 水 → 冰
- 岩浆 → 黑曜石
- 其他液体固化

**能源消耗**: 中等
**处理速度**: 快

## 🌾 农业与生物机器

### 自动喂食器 (Auto Feeder)

自动为动物提供食物。

**特点**:
- 支持所有家畜
- 自动繁殖控制
- 减少人工操作

**使用方法**:
1. 放置在家畜附近
2. 添加食物到内部存储
3. 设置繁殖数量上限
4. 启动机器

**能源消耗**: 低
**存储容量**: 一组食物

### 动物生长加速器 (Animal Growth Accelerator)

加速幼年动物的成长。

**特点**:
- 缩短 90% 成长时间
- 支持所有动物
- 需要能源供应

**使用方法**:
1. 将幼年动物围在加速器范围内
2. 启动机器
3. 等待动物快速成长

**范围**: 5x5 区域
**能源消耗**: 中等
**加速倍率**: 10 倍

### 作物生长加速器 (Crop Growth Accelerator)

加速作物生长的设备。

**特点**:
- 加速所有作物生长
- 大范围覆盖
- 自动收集成熟作物（高级版）

**使用方法**:
1. 放置在农田中央
2. 启动机器
3. 享受丰收

**范围**: 9x9 区域
**加速倍率**: 5-8 倍
**能源消耗**: 中等

### 堆肥机 (Food Composter)

将有机物转化为堆肥。

**输入材料**:
- 种子
- 树叶
- 腐肉
- 其他有机物

**输出产品**:
- 堆肥（肥料）
- 蘑菇（几率）

**处理时间**: 30 秒/个
**能源消耗**: 低

### 经验收集器 (XPCollector)

自动收集玩家掉落的经验值。

**特点**:
- 存储大量经验
- 可为物品附魔
- 支持经验提取

**使用方法**:
1. 放置在刷怪塔或挖矿区域
2. 自动吸收经验球
3. 使用经验瓶提取

**存储容量**: 100,000 XP
**能源消耗**: 低

## 📦 货物运输系统

### 货物输入节点 (Cargo Input Node)

自动输入物品到机器。

**特点**:
- 从容器提取物品
- 可设置过滤
- 支持红石控制

**配置选项**:
- 物品过滤（白名单/黑名单）
- 数量限制
- 提取速度

**传输速度**: 1 物品/秒（基础）
**能源消耗**: 低

### 货物输出节点 (Cargo Output Node)

从机器输出物品到容器。

**特点**:
- 向容器输出物品
- 可设置优先级
- 支持堆叠控制

**配置选项**:
- 输出方向
- 物品筛选
- 堆叠大小

**传输速度**: 1 物品/秒（基础）
**能源消耗**: 低

### 高级货物输出节点 (Advanced Cargo Output Node)

智能物品分配系统。

**高级功能**:
- 多方向输出
- 智能分配
- 优先级排序
- 条件过滤

**使用方法**:
1. 放置在机器输出面
2. 配置输出规则
3. 连接多个目标容器
4. 设置优先级

**传输速度**: 2 物品/秒
**能源消耗**: 中等

### 货物合成节点 (Cargo Crafting Node)

自动执行合成配方。

**特点**:
- 集成输入和合成
- 支持复杂配方
- 可连接货物网络

**使用方法**:
1. 设置合成配方
2. 连接输入节点
3. 连接输出节点
4. 启动系统

**合成速度**: 取决于配方
**能源消耗**: 中等

## 🔧 特殊用途机器

### 自动铁砧 (Auto Anvil)

自动修复工具和武器。

**特点**:
- 无需经验值
- 自动修复
- 支持附魔物品

**使用方法**:
1. 放入损坏的物品
2. 放入修复材料（相同物品）
3. 启动机器
4. 取出修复后的物品

**修复成本**: 能源 + 材料
**处理时间**: 5 秒

### 自动附魔台 (Auto Enchanter)

自动为物品附魔。

**特点**:
- 无需青金石
- 可控制附魔结果
- 支持高级附魔

**使用方法**:
1. 放入待附魔物品
2. 放入附魔书
3. 启动机器
4. 取出附魔物品

**能源消耗**: 高
**处理时间**: 10 秒

### 自动分解台 (Auto Disenchanter)

移除物品的附魔并返还材料。

**特点**:
- 移除不需要的附魔
- 返还部分材料

**返还物品**:

- 附魔书（几率）

## 💡 机器使用技巧

### 能源管理

1. **优先级排序**
   - 关键机器优先供电
   - 非关键机器错峰运行
   - 使用能源存储缓冲

2. **效率优化**
   - 批量处理物品
   - 避免机器空转
   - 定期维护

### 自动化设计

1. **输入输出系统**
   - 使用货物节点连接
   - 设置合理的过滤器
   - 避免物品堵塞

2. **连锁反应**
   - 机器串联工作
   - 中间产品自动转运
   - 成品自动存储

### 故障排查

1. **机器不工作**
   - 检查电源连接
   - 确认输入物品正确
   - 查看输出槽是否为空

2. **物品卡住**
   - 检查货物节点设置
   - 确认管道连接正确
   - 重启机器

3. **能源不足**
   - 增加发电设备
   - 减少同时运行的机器
   - 使用能源存储
