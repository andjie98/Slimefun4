# Slimefun4 中文 Wiki

欢迎来到 **Slimefun4 中文定制版** 的官方文档站点！

## 🎯 简介

**Slimefun4** 是一个革命性的 Minecraft 插件，它将你的服务器变成一个功能齐全的模组包，而无需安装任何客户端模组！

### 版本信息
- **当前版本**: 4.1.15
- **适用版本**: Minecraft 1.12.2
- **许可证**: GNU GPLv3

### 核心特性

#### 🔧 自动化机器体系
- 能源生成设备
- 材料加工机器
- 农业自动化

#### ⚡ 能量网络
- 完整的能源存储和传输

## 📥 快速开始

### 安装步骤

1. **下载插件**
   - 从 [Releases](https://github.com/andjie98/Slimefun4/releases) 下载
   - 下载 CS-CoreLib v1.5.18

2. **安装文件**
   ```
   server/
   └── plugins/
       ├── Slimefun v4.1.15.jar
       └── CS-CoreLib v1.5.18.jar
   ```

3. **启动服务器**
   - 首次运行自动生成配置
   - 检查控制台错误信息

### 游戏入门

1. **加入服务器** - 自动获得 Slimefun 指南
2. **查看指南** - 右键点击指南查看物品
3. **开始研究** - 消耗经验解锁配方
4. **制作工具** - 从基础工具开始
5. **建造机器** - 建立自动化生产线
